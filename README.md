## Install

```sh
make
```

## Usage

```sh
./zappy_server
```

You can access the server on `zappy.marc0.fr`. The port used is `2142`

## Run tests

```sh
make -C server tests_run
```

The coverage is available [here](https://zappy.marc0.fr)
